# Professional Restaurant Menu Design Guidelines
## AI-Powered PDF Menu Generator Standards

### 1. Visual Hierarchy & Structure

#### **Category Order (Strict Priority)**
1. **Starters/Appetizers** - Light dishes, soups, salads
2. **Main Courses** - Primary food items, entrees
3. **Desserts** - Sweet endings, pastries
4. **Soft Drinks** - Non-alcoholic beverages, juices, sodas
5. **Hot Beverages** - Coffee, tea, hot chocolate
6. **Cocktails** - Mixed alcoholic drinks
7. **Spirits** - Straight alcoholic beverages
8. **Wines** - Red, white, rosé, sparkling
9. **Others** - Any remaining items

#### **Page Layout Standards**
- **Format**: A4 Portrait (210mm × 297mm)
- **Margins**: 20mm on all sides
- **Header Space**: 60mm for branding
- **Footer Space**: 15mm for contact info
- **Content Area**: 202mm × 222mm
- **Column Layout**: Single column with right-aligned prices

### 2. Typography Pairing Rules

#### **Premium Font Combinations**
1. **Minimal Theme**: 
   - Titles: Inter (Bold, 24pt)
   - Body: Inter (Regular, 12pt)
   - Accent: Inter (Medium, 14pt)

2. **Rustic Theme**:
   - Titles: Playfair Display (Bold, 24pt)
   - Body: Poppins (Regular, 12pt)
   - Accent: Poppins (Medium, 14pt)

3. **Luxury Theme**:
   - Titles: Cormorant Garamond (Bold, 26pt)
   - Body: Lato (Regular, 12pt)
   - Accent: Lato (Medium, 14pt)

4. **Modern Theme**:
   - Titles: Montserrat (Bold, 24pt)
   - Body: Open Sans (Regular, 12pt)
   - Accent: Open Sans (Medium, 14pt)

#### **Font Size Hierarchy**
- **Restaurant Name**: 24-26pt, bold
- **Category Headers**: 16-18pt, bold, uppercase
- **Item Names**: 12-14pt, regular
- **Descriptions**: 10-11pt, italic
- **Prices**: 12-14pt, bold
- **Contact Info**: 9-10pt, regular

### 3. Category Grouping & Icons

#### **Category Icons (SVG)**
- **Starters**: 🥗 (Leaf icon)
- **Main Courses**: 🍽️ (Plate icon)
- **Desserts**: 🍰 (Cake icon)
- **Soft Drinks**: 🥤 (Glass icon)
- **Hot Beverages**: ☕ (Coffee cup icon)
- **Cocktails**: 🍸 (Cocktail glass icon)
- **Spirits**: 🥃 (Whiskey glass icon)
- **Wines**: 🍷 (Wine glass icon)

#### **Category Separation**
- **Visual Separators**: Elegant lines or decorative elements
- **Spacing**: 24pt between categories
- **Background**: Subtle color tinting for category headers
- **Typography**: Uppercase, bold category names

### 4. Spacing & Layout Rules

#### **Consistent Spacing**
- **Category Spacing**: 24pt between categories
- **Item Spacing**: 8-10pt between items
- **Description Spacing**: 4pt below item name
- **Price Alignment**: Right-aligned, 15mm from right margin
- **Line Height**: 1.4x for descriptions, 1.6x for categories

#### **Price Formatting**
- **Dotted Leaders**: Connect item names to prices
- **Currency**: Use $ symbol
- **Decimal Places**: Always show 2 decimal places
- **Alignment**: Right-aligned column
- **Spacing**: Consistent 15mm from right edge

### 5. Brand Integration Rules

#### **Logo Integration**
- **Position**: Centered in header area
- **Size**: Max 40mm height, maintain aspect ratio
- **Background**: Transparent or subtle background
- **Spacing**: 10mm below logo to restaurant name

#### **Color Integration**
- **Primary Brand Color**: Use for category headers and accents
- **Secondary Color**: Use for subtle highlights
- **Text Colors**: High contrast for readability
- **Background**: Clean white or subtle tint

#### **Typography Branding**
- **Restaurant Name**: Use brand font if available
- **Consistency**: Maintain brand typography throughout
- **Hierarchy**: Clear visual hierarchy with brand elements

### 6. AI-Powered Theme System

#### **Theme Categories**
1. **Minimal Theme**:
   - Clean, simple design
   - Lots of whitespace
   - Subtle typography
   - Minimal decorative elements

2. **Rustic Theme**:
   - Warm, organic feel
   - Serif fonts for titles
   - Natural color palette
   - Hand-drawn style elements

3. **Luxury Theme**:
   - Elegant, sophisticated design
   - Premium typography
   - Gold/silver accents
   - Refined spacing

4. **Modern Theme**:
   - Contemporary, clean design
   - Sans-serif fonts
   - Bold typography
   - Geometric elements

#### **AI Theme Selection Logic**
- **Restaurant Type**: Fine dining → Luxury, Casual → Modern
- **Cuisine Style**: Italian → Rustic, Asian → Minimal
- **Target Audience**: Business → Modern, Family → Rustic
- **Brand Colors**: Warm colors → Rustic, Cool colors → Modern

#### **Item Structure**

#### **Emphasis Techniques**
- **Signature Dishes**: Bold item name + special icon
- **Premium Items**: Slight price emphasis + star icon
- **Chef's Specials**: Small crown icon or asterisk (*)
- **Seasonal Items**: Leaf icon or seasonal indicator

### 8. Content Guidelines

#### **Item Descriptions**
- **Maximum Length**: 2 lines maximum
- **Style**: Italic, smaller font
- **Content**: Key ingredients, preparation method
- **Language**: Clear, appetizing descriptions

#### **Category Naming**
- **Food Categories**: Use existing database categories
- **Drink Categories**: Distinguish clearly between alcoholic/non-alcoholic
- **Consistency**: Maintain database category names

#### **Price Presentation**
- **Format**: $XX.XX (always 2 decimal places)
- **Alignment**: Right-aligned column with dotted leaders
- **Consistency**: Same format throughout

### 9. Quality Standards

#### **Readability**
- **Contrast**: High contrast text on white background
- **Line Height**: 1.4x for descriptions, 1.6x for categories
- **Paragraph Spacing**: Consistent throughout

#### **Professional Appearance**
- **Clean Layout**: Minimalist design approach
- **Balanced Whitespace**: Proper spacing between elements
- **Consistent Alignment**: Left-aligned text, right-aligned prices

#### **Print-Ready Output**
- **Resolution**: 300 DPI minimum
- **Color Mode**: CMYK for professional printing
- **File Format**: PDF/A-1b for archival quality

### 10. AI Classification Rules

#### **Food vs. Beverage Detection**
- **Food Keywords**: dish, plate, meal, food, cuisine
- **Beverage Keywords**: drink, beverage, cocktail, wine, spirit, beer
- **Context Analysis**: Use item names and descriptions

#### **Alcoholic vs. Non-Alcoholic**
- **Alcoholic Indicators**: wine, beer, spirit, cocktail, vodka, whiskey
- **Non-Alcoholic Indicators**: juice, soda, water, coffee, tea
- **Default Classification**: When uncertain, classify as "Others"

#### **Category Assignment**
- **Use Database Categories**: Respect existing category structure
- **Smart Grouping**: Group similar items within categories
- **Fallback**: If no category exists, use "Others"

### 11. Technical Constraints

#### **Data Source**
- **Dynamic Fetching**: All items from restaurant database
- **No Hardcoding**: No static menu items
- **Real-time Data**: Always use current menu data

#### **Design Limitations**
- **Light Theme Only**: No dark mode support
- **Minimalist Approach**: No experimental layouts
- **Consistent Style**: Maintain uniform appearance

#### **Performance**
- **Generation Time**: Maximum 30 seconds
- **File Size**: Optimize for web download
- **Compatibility**: Works across all modern browsers

### 12. Export Options

#### **PDF Settings**
- **Page Size**: A4 Portrait
- **Quality**: High resolution (300 DPI)
- **Compression**: Optimized for web and print
- **Metadata**: Include restaurant name and generation date

#### **User Controls**
- **Category Reordering**: Allow manual category arrangement
- **Item Reordering**: Allow manual item arrangement within categories
- **Preview**: Show preview before final export
- **Download**: Direct download link
